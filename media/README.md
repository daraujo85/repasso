# Pasta de Mídia

Esta pasta contém os arquivos de mídia usados nos exercícios do Repaso.

## Estrutura

- `audio/` - Arquivos de áudio (formato recomendado: MP3)
- `images/` - Imagens (formatos recomendados: JPG, PNG, WebP)
- `videos/` - Vídeos (formatos recomendados: MP4, WebM)

## Como usar nos exercícios

### Áudio
```json
{
  "audio": "media/audio/nome-do-arquivo.mp3",
  "altImagem": "Descrição do áudio"
}
```

### Imagem
```json
{
  "imagem": "media/images/nome-do-arquivo.jpg",
  "altImagem": "Descrição da imagem"
}
```

### Vídeo
```json
{
  "video": "media/videos/nome-do-arquivo.mp4",
  "altImagem": "Descrição do vídeo"
}
```

## Notas

- Mantenha os nomes dos arquivos descritivos e em minúsculas
- Use hífens ao invés de espaços nos nomes dos arquivos
- Otimize imagens e vídeos para web quando possível
- Para áudios, use qualidade adequada mas mantenha o tamanho do arquivo razoável
